# tsconfig.json

target
・変換する JS の ES バージョン。

lib
・型定義ファイル。
・target に指定している ES バージョンの型定義ファイルは自動で含まれている。

module
・トランスパイル後の JS のモジュールタイプ
・構文を変換するだけで、実際に「"./hello"」が存在するかどうかまでとかは関係ない。

moduleResolution
・実際にトランスパイルするときに、"./hello"を参照するときに拡張子を補完するかどうか等を
制御するオプション。

node -> cjs と同じ制御となる=「.js」「.cjs」を自動補完して探しに行く。あれば OK。
